  • TestNG 6.8.6.20130914_0724
  • VJET 0.10.0.201309201624 (Experimental/Incubating)

Note : Google Plugin for Eclipse 3.4.2 for Kepler (e43) doesn't update cleanly from the previous 3.2.x versions (Juno/e42). If you update you'll most probably see the remediation screen asking you to uninstall e42 before installing e43.

Install via Drag N'Drop

You can drag a link into the JBoss Central view to cause Central connector(s) to be installed into Eclipse or JBDS.

  • Drag this link into Eclipse to install Spring IDE and m2e-wro4j

Supported link formats :

  • http://download.jboss.org/jbosstools/central/install?connectors=comma-separated-list-of-connector-ids
  • https://devstudio.jboss.com/central/install?connectors=comma-separated-list-of-connector-ids
